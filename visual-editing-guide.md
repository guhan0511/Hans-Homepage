# 浏览器可视化编辑指南

## 1. 查看作品

您可以通过以下地址在浏览器中查看您的设计师作品集：
**http://localhost:8000**

## 2. 浏览器可视化编辑方法

### 方法一：使用浏览器开发者工具（推荐）

大多数现代浏览器（Chrome、Firefox、Safari）都内置了强大的开发者工具，可以让您实时编辑和预览网页内容和样式。

#### 步骤：

1. **打开开发者工具**：
   - Chrome/Safari：右键点击页面任意位置，选择「检查」(Inspect) 或按 `Command + Option + I` (Mac) / `Ctrl + Shift + I` (Windows)
   - Firefox：右键点击页面任意位置，选择「检查元素」或按 `Command + Option + C` (Mac) / `Ctrl + Shift + C` (Windows)

2. **定位元素**：
   - 使用开发者工具左上角的「选择元素」图标（箭头），然后点击页面上要编辑的元素
   - 或者直接在「Elements」面板中浏览HTML结构找到要编辑的元素

3. **编辑内容**：
   - 在「Elements」面板中，双击要修改的文本内容，直接输入新内容
   - 修改后会立即在页面上看到效果

4. **编辑样式**：
   - 在「Elements」面板中选择元素后，右侧会显示「Styles」面板
   - 可以修改现有CSS属性值，或点击「+」添加新样式
   - 修改后会立即在页面上看到效果

5. **复制修改后的代码**：
   - 编辑完成后，右键点击修改的元素，选择「Copy」→「Copy element」
   - 然后将复制的代码粘贴回您的本地HTML文件中保存

### 方法二：直接编辑本地文件

1. 使用您喜欢的代码编辑器（如VS Code、Sublime Text等）打开项目文件夹
2. 编辑 `index.html` 或其他HTML文件
3. 保存文件后，刷新浏览器页面（按 `Command + R` 或 `F5`）即可看到更改

## 3. 编辑提示

### 编辑个人信息

- **姓名**：修改第12行的 `<title>` 标签和第118行的导航栏名称
- **个人简介**：修改第153-155行的Hero区域简介文本
- **技能**：修改第316-329行的技能标签
- **经验和教育**：修改第336-370行的经验和教育信息

### 修改样式

- **颜色**：修改第17-22行的Tailwind配置中的颜色值
- **字体**：修改第23-25行的字体配置
- **动画效果**：修改第35-111行的自定义CSS样式

### 添加新作品

1. 在 `works/` 目录下创建新的HTML文件（如 `work7.html`）
2. 参考现有作品文件的结构编写内容
3. 在 `index.html` 的作品网格（第182-290行）中添加新的作品卡片

## 4. 保存更改

浏览器开发者工具中的编辑是临时的，刷新页面后会恢复原状。要永久保存更改：

1. 在开发者工具中完成可视化编辑
2. 复制修改后的代码
3. 粘贴到本地HTML文件中并保存
4. 刷新浏览器确认更改已保存

## 5. 预览不同设备效果

使用浏览器开发者工具的「响应式设计模式」可以预览网站在不同设备上的效果：

- Chrome/Safari：点击开发者工具左上角的手机/平板图标
- Firefox：按 `Command + Option + M` (Mac) / `Ctrl + Shift + M` (Windows)
- 选择预设设备或自定义尺寸查看响应式效果

## 6. 调试技巧

- 使用「Console」面板查看JavaScript错误
- 使用「Network」面板检查资源加载情况
- 使用「Application」面板管理Cookie和本地存储

通过以上方法，您可以轻松地在浏览器中可视化编辑您的设计师作品集，并实时查看效果！
